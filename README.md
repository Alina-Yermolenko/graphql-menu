# Project Title

"GraphQL"

## Getting Started

1. Clone the repo
    ```
    git clone https://github.com/Alina-Yermolenko/graphql-menu.git
    ```

1. `cd graphql-menu`
1. Install dependencies and start the development server: `npm i && npm start`

The menu should now be available at http://localhost:3000.
